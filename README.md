# 公開資訊觀測站爬蟲

## Introduce
目前只支援Chrome，請至[ChromeDriver](https://chromedriver.chromium.org/downloads)下載相同版本Driver。

## Usage
```commandline
git clone https://github.com/ytforfun/mops_twse_crawler_python.git
cd mops_twse_crawler_python
python mops.py
```
